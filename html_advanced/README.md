## 📌 Guidelines HTML

✔️ Bonnes pratiques :

* Code lisible et bien indenté
* Utilisation des balises sémantiques
* Respect des standards W3C
* Fermeture des balises
* Noms explicites

---

## 🧱 Skeleton HTML5

```html
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Mon site</title>
</head>
<body>

</body>
</html>
```

---

## 🏷️ Balises sémantiques

| Balise  | Rôle                |
| ------- | ------------------- |
| header  | En-tête             |
| main    | Contenu principal   |
| footer  | Pied de page        |
| article | Contenu indépendant |
| section | Regroupement        |
| nav     | Navigation          |
| aside   | Contenu secondaire  |

---

## 📦 div vs span

| Élément | Type   | Usage             |
| ------- | ------ | ----------------- |
| div     | Bloc   | Structure         |
| span    | Inline | Styliser du texte |

---

## 🧠 Valeur sémantique

Les balises sémantiques améliorent :

* 📈 SEO
* ♿ Accessibilité
* 🧩 Compréhension du code

## 🔢 Headings

```html
<h1>Titre principal</h1>
<h2>Sous-titre</h2>
```

✔️ Règles :

* 1 seul `<h1>`
* Respect de la hiérarchie

## 📋 Listes

### Liste non ordonnée

```html
<ul>
  <li>Item</li>
</ul>
```

### Liste ordonnée

```html
<ol>
  <li>Item</li>
</ol>
```
